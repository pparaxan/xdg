### `xdg`

This repository provides both header and source file of the translated XML protocol of `shell` and `decoration`. This repository solely exists because both is needed and used for the Linux side of things for [Quark](https://codeberg.org/pparaxan/Quark), an vulkan-based GUI toolkit used to build fast, native applications.
